# FairSlot (Utility Scheduler)

A shared utility workspace for organizations and residential communities to seamlessly manage resources like parking, halls, EV chargers, and more.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Auth:** JWT & Google OAuth

## Quick Start
```bash
# Start backend
cd server
npm install
npm run dev

# Start frontend
cd client
npm install
npm run dev
```
